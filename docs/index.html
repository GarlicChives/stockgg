  • DRAM、NAND Flash合約價持續上漲，DDR4、DDR5、HBM需求同步激增。
  • 美光、三星、SK海力士等原廠積極擴產，但無塵室空間及HBM產能消耗限制供給。
  • 代理式AI興起，帶動通用伺服器及非HBM記憶體需求，傳統DDR成長幅度可能超越HBM。
  • 三星記憶體罷工事件可能進一步影響供給，對台廠有利。
