綜合多空判斷

  • 短期(1-2週)：偏多 — AI相關企業財報與展望持續優於預期，帶動科技股上漲。中東地緣風險鈍化，市場回歸基本面定價。
  • 中期(1-3月)：偏多 — 代理式AI需求激增，推升半導體設備、記憶體及先進製程需求。台灣出口數據強勁，顯示經濟基本面穩固。
  • 長期(3-12月)：偏多 — AI產業的結構性成長趨勢明確，帶動相關供應鏈長期投資。記憶體產業復甦週期確立，有助於整體半導體景氣。
  • 關鍵風險： 聯準會內部意見分歧加劇可能導致政策溝通混亂；中東局勢若再度惡化，油價突破關鍵價位恐引發通膨擔憂；消費性電子需求復甦不如預期，可能限制部分半導體產品漲價空間。
